# Trading-Test
Сервис для тестирования моделей по прогнозированию фондовых индексов


**Описание проекта**:
Сервис должен позволять осуществлять тестирование моделей прогнозирования динамики фондовых индексов. 

**Заказчик**:
Центр аналитики динамических процессов и систем СПбГУ

---

## Запуск сервиса
1) Клонировать данный репозиторий к себе
### Для локального запуска (без Docker):
2) Установить **node.js**, запустить npx.cmd по пути _/Trading-Test/frontend/_ запустить команду:
```bash
npm run dev
```
3) Установить **Python 3.9+**, создать виртуальное окружение
```bash
python -m venv venv
```
запустить его (Windows)
```bash
.\venv\Scripts\activate
```
запустить его (Linux)
```bash
source ./venv/bin/activate
```
и запустить команду
```bash
pip install -r requirements.txt
```
из каждой серверной папки:
- /Trading-Test/server/dataloader/
- /Trading-Test/server/ml-testing-service/
- /Trading-Test/server/db-service/
4) Запустить серверы из вышеперечисленных папок (в отдельных терминалах):
  - dataloader.py
  - model_tester.py
  - server_db.py
5) Можно использовать сервис по адресу localhost:5173.
При необходимости можно заменить URL сервисов в файле frontend/.env.local
### Для запуска через Docker:
2) Запуск
```bash
docker-compose up --build
```
3) Остановка
```bash
docker-compose down
```

## Дополнительно
В веб-сервисе вы можете увидеть несколько функциональных окон, среди которых - datasets, expirements, models.
Для инициации эксперимента вам необходимо:
1) Создать датасет в окне datasets, выбрав определенный тикер и интервал, а также сохранив его с помощью кнопки New.
2) Загрузить модель в окне models, можно выбрать веса из папки /server/ml-testing-service/examples. Там же можно найти .json файл с параметрами препроцессинга, которые нужно вписать в окне загрузки модели.
3) В окне expirements можно создать новый эксперимент, выбрав датасет и модель.
